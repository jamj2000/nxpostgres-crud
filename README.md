## Crear base de datos y lanzar aplicación

**Pasos**

1. Entrar en la siguiente carpeta:

```sh
cd src/database
```

2. Editar el archivo `config.mjs`:

```sh
nano  config.mjs
```

3. Volver a la carpeta raíz del proyecto:

```
cd  ../..
```


4. Ejecutar script:

```sh
npm run seed
```

4. Lanzar aplicación:

```sh
npm run dev
```
